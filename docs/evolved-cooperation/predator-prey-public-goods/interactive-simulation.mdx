This page integrates the browser replay of the predator-prey public-goods run directly into the site page.

The replay is interactive in the limited sense that you can:

- play and pause the exported run,
- restart from the beginning,
- change playback speed,
- scrub through sampled frames with the frame slider.

It is still a replay, not a live browser execution of the Python model.

[Open Source Module](https://github.com/doesburg11/EvolvedCooperation/tree/main/predpreygrass_public_goods)
